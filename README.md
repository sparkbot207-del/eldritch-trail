## Features

- **6 Unique Investigators** - Each with different starting stats and abilities
- **Resource Management** - Balance Sanity, Supplies, and Fuel
- **Pursuit Mechanic** - They're coming for you. Don't let them catch up.
- **Random Events** - Choices with consequences
- **Tappable Hotspots** - Explore each scene for hidden details
- **Multiple Endings** - Madness, capture, death... or escape?

## Controls

- **DRIVE** - Travel to the next location (costs 1 Fuel)
- **SEARCH** - Search the current location for supplies
- **REST** - Recover Sanity (costs 2 Supplies, increases Pursuit)
- **PARTY** - View your traveling companions

Tap glowing hotspots in scenes to examine your surroundings.
